# PC Gamer Cows

This resource pack brings back the legendary PC Gamer cows from the [Minecraft PC Gamer demo](https://minecraft.wiki/w/Java_Edition_Demo_(PC_Gamer)) that came with PC Gamer issue 214¹. With faithfully restored cow and baby cow textures that will make your videos and gameplay... exactly the same, but with a special treat for attentive viewers I guess.

## Video usage guidelines

If you use this resource pack in a video that will be publicly viewable (e.g. posted on tik-tok or youtube) then you have to:

- Provide links to the modrinth, curseforge, and github pages.
- NOT take attribution for the resource pack.

## Building the resource pack

This pack is built using packsquash, use the following command to build:

```bash
./[INSERT_PATH_TO_PACKSQUASH_EXECUTABLE_HERE] packSquash.toml
```
